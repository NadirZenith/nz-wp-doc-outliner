Adds a button in wp admin top bar to show the document outline

== Description ==

View you document ouline while your developing

== Installation ==

1. Upload `nz-wp-doc-outliner` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Access any page and click on the top button `outline`
